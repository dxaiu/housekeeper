# 生活管家项目简介

## 项目概述
生活管家是一款面向个人用户的生活缴费管理应用，旨在帮助用户方便地记录、管理和分析日常生活中的各项缴费支出，包括水费、电费、燃气费、物业费和停车费等。

## 核心功能

### 1. 缴费记录管理
- 支持添加、编辑和删除缴费记录
- 按年份和月份筛选记录
- 按费用类型分类查看

### 2. 数据统计与分析
- 月度费用统计
- 年度费用分析
- 费用趋势图表展示
- 支持按年份和月份筛选数据
- 支持多种图表类型（柱状图、折线图、曲线图）

### 3. 缴费周期管理
- 物业费缴费周期设置
- 停车费缴费周期设置
- 剩余天数提醒

### 4. 数据导出与备份
- 导出所有数据为Excel文件
- 备份数据到本地
- 从本地恢复数据
- 支持H5浏览器和HBuilderX安卓app环境

### 5. 个性化设置
- 首页模块显示开关
- 单价设置

## 技术特点

### 前端技术栈
- HTML5 + CSS3 + JavaScript
- Tailwind CSS 用于响应式布局
- Chart.js 用于数据可视化
- SheetJS (xlsx) 用于Excel导出
- Font Awesome 用于图标

### 数据存储
- IndexedDB 用于本地数据存储
- localStorage 用于配置信息存储

### 应用特性
- PWA 支持，可离线使用
- 响应式设计，适配移动端
- 现代化UI，符合iOS风格
- 支持多平台（H5浏览器、安卓app）

## 目标用户
- 个人用户：需要管理日常缴费记录的家庭用户
- 租户：需要记录和管理租房期间的各项费用
- 小型企业：需要简单管理办公场所的缴费支出

## 项目价值
- 帮助用户更好地了解和管理个人财务
- 提供数据可视化分析，帮助用户优化消费
- 减少缴费逾期风险，通过缴费周期提醒
- 提供数据备份和导出功能，确保数据安全
